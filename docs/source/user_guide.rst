Guide d’utilisation
===================

Navigation
----------

- Accédez à la liste des profils.
- Cliquez sur un profil pour afficher son détail.
- Accédez à la liste des locations.
- Cliquez sur une location pour afficher plus d'informations.

Cas d’utilisation
-----------------

Afficher un profil
~~~~~~~~~~~~~~~~~~

1. Aller sur la page ``/profiles/``.
2. Choisir un utilisateur.
3. Voir ses informations (ville préférée, identifiant).

Afficher une location
~~~~~~~~~~~~~~~~~~~~~

1. Accéder à ``/lettings/``.
2. Sélectionner une location.
3. Visualiser ses informations (adresse, titre).
