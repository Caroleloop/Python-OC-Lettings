Interfaces de programmation
===========================

Routes principales
------------------

Profils
~~~~~~~

- ``/profiles/`` : liste des profils
- ``/profiles/<id>/`` : détail d’un profil

Locations
~~~~~~~~~

- ``/lettings/`` : liste des locations
- ``/lettings/<id>/`` : détail d’une location

Format des réponses
-------------------

Pages HTML rendues via Django templates.
