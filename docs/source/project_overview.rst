
Description du projet
=====================

Orange County Lettings est une application web construite avec Django qui
permet d'afficher des profils utilisateurs ainsi que des locations immobilières.

Objectifs
---------

- Offrir un site simple pour consulter des profils et des locations.
- Servir de projet de démonstration pour les bonnes pratiques Django.
- Fournir une base pour un pipeline complet incluant tests, linting et déploiement.

Fonctionnalités principales
---------------------------

- Liste des profils utilisateurs
- Détail d’un profil
- Liste des locations immobilières
- Détail d’une location

Architecture générale
---------------------

- Framework : Django
- Base de données locale : SQLite3
- Déploiement : Render 
- Suivi d’erreurs : Sentry
